# Hiring AI Hub Winter 2026

## 第 1 页

Join Algonquin’s Applied Research AI Accelerator Hub
加入阿冈昆学院应用研究 AI 加速器中心

Algonquin College’s AI Accelerator Hub is now recruiting for student positions for Winter 2026.
阿冈昆学院 AI 加速器中心现正招聘 2026 年冬季学生岗位。

FT/PT Winter Positions January 12 – April 17, 2026:
全职/兼职冬季职位，时间为 2026 年 1 月 12 日至 4 月 17 日：

This opportunity is available only to current Algonquin College students and is for full-time 35/ week (co-op eligible) or part-time max. 20 hours/week positions at the Ottawa Campus during the Winter term. All positions are paid at Ottawa’s Living Wage ($23.40/Hour).
此机会仅面向阿冈昆学院在读学生，冬季学期在渥太华校区提供每周 35 小时的全职（符合带薪实习资格）岗位或每周最多 20 小时的兼职岗位。所有岗位均按渥太华生活工资标准支付（23.40 加元/小时）。

Desired Skills:
岗位技能要求：

- Strong background as a generalist web developer (both front and back-end)
  - 具备扎实的全栈 Web 开发背景（前端与后端）
- Experience building frontend component-based architecture
  - 拥有构建前端组件化架构的经验
- Understanding of web technologies, protocols and systems (i.e., HTTP, REST, APIs, JSON, CMS, CRMS, Mobile Web, etc.)
  - 理解 Web 技术、协议与系统（如 HTTP、REST、API、JSON、CMS、CRMS、移动 Web 等）
- Knowledge and experience using the GIT version control
  - 熟悉并具备使用 Git 版本控制的经验
- Experience with Linux based systems
  - 拥有使用 Linux 系统的经验
- Experience with web accessibility standards
  - 熟悉 Web 可访问性标准
- Experience with JavaScript, React & NPM
  - 具备 JavaScript、React 与 NPM 的使用经验
- Experience with Python and frameworks like Django, Flask or similar concepts
  - 具有 Python 及 Django、Flask 等相关框架的经验

Bonus Skills:
附加技能：

- Experience with AI implementation frameworks
  - 拥有使用 AI 落地框架的经验
- Experience with agentic frameworks
  - 熟悉智能体相关框架
- Experience building LLM-based solution on AWS, Azure or GCP
  - 具备在 AWS、Azure 或 GCP 上构建基于大语言模型解决方案的经验
- Experience in building and deploying code in Python, SQL and or major programming languages with at least one major Cloud platform (Aws, Azure, GCP)
  - 在至少一种主流云平台（AWS、Azure、GCP）上使用 Python、SQL 或其他主要编程语言构建并部署代码的经验
- Exposure to LLMS such as OpenAI, Claude, Mistral or LLaMA
  - 熟悉 OpenAI、Claude、Mistral 或 LLaMA 等大语言模型
- Experience with LangChain, Hugging Face Transformers or similar frameworks
  - 拥有 LangChain、Hugging Face Transformers 等类似框架的经验

Soft Skills:
软技能：

- The ability to juggle multiple tasks at once
  - 能够同时处理多项任务
- Ability to troubleshoot and debug your own/ third-party code
  - 能够排查并调试自身或第三方代码
- You are naturally curious and eager to learn about AI/LLM-powered interfaces and agentic workflows
  - 天性好奇，渴望学习基于 AI/LLM 的界面与智能体流程
- Excellent communication skills with client-facing experience
  - 具备优秀的沟通能力及客户交流经验

To Apply Sen d CV + Cover letter highlighting match to the skills noted above to AIHub@AlgonquinCollege.com by 1 1:59 pm Sunday November 30, 2025
申请方式：请于 2025 年 11 月 30 日（星期日）晚上 11:59 前，将简历和求职信（强调与上述技能的匹配度）发送至 AIHub@AlgonquinCollege.com

我们认识到在各项工作中持续提升公平性、多样性与包容性的必要性，强烈鼓励、欢迎并珍视女性、原住民、残障人士、可见少数族裔/种族化群体成员以及 LGBTQ2+ 群体的申请。
